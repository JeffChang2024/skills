# Bilibili AI Video Prompt Analyzer

Extract/reverse-engineer prompts from Bilibili AI-generated videos for imitation. Automatically detects scene changes, splits into storyboard shots, and generates per-shot prompts.

## Workflow

1. **Get video info**: Download the video from Bilibili URL
2. **Extract frames**: Extract frames at configurable FPS (default: 1 frame/second)
3. **Scene change detection**: Group frames into shots based on visual change magnitude
4. **Visual analysis**: Analyze each shot to identify style, subjects, colors, composition changes
5. **Generate prompts**:
   - Overall style prompt for the whole video
   - Per-shot prompts detailing the evolution
6. **Output**: Complete storyboard with full prompt + per-shot breakdown

## When to Use

✅ Use this skill when:
- You have a Bilibili link to an AI-generated video
- You want to imitate the style and generate similar videos
- You need to reverse-engineer the prompt from the video content
- You want a complete storyboard breakdown with per-frame/per-shot prompts

❌ Don't use when:
- The prompt is already in the video description (use agent-reach to extract directly)
- It's a human-shot video not generated by AI

## Sampling Strategy

| FPS | Use Case | Recommendation |
|-----|----------|----------------|
| 0.2 | Long video (>5min), big scene changes | Only key frames |
| 1 | Short video (<1min) | **Default**, one frame per second |
| 2-5 | Detailed motion analysis | Slow morphing animations |

**Best practice**: 1fps for short videos (<1min) gives good granularity without redundancy. Most AI video evolution changes gradually, 1s interval captures the arc well.

## Requirements

- `ffmpeg` installed (for frame extraction)
- Multimodal LLM support (for visual frame analysis)
- `yt-dlp` for video download

## Usage Example

```
Given Bilibili URL: https://b23.tv/CKF9Kb8
1. Download video → /tmp/video.mp4
2. Extract 1fps → /tmp/frames/frame%04d.jpg
3. Detect scene changes → 5 shots detected
4. Analyze each shot → get visual description per shot
5. Output → overall style + per-shot prompt evolution
```

## Scripts

- `scripts/download-bilibili.sh <url> <output-dir>` - Download video from Bilibili URL
- `scripts/extract-frames.sh <video-file> <output-dir> [fps]` - Extract frames from video

## Output Format

The final output includes:

**Overview**
1. **Overall style prompt** - Master prompt for the whole video
2. **Global analysis** - Art style, camera, colors, pacing, aspect ratio
3. **Technical suggestions** - Model recommendations, settings

**Storyboard Breakdown** (per shot)
- Shot time range (start-second → end-second)
- Key visual changes from previous shot
- Shot-specific prompt extension
- Visual description (composition, colors, focal point)

**Arc Summary**
- Narrative evolution from start to finish
- Recommended prompt composition for reproduction
