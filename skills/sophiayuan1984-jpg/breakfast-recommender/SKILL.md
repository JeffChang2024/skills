# Breakfast Recommender

智能早餐推荐助手，根据你冰箱里的食材，每天为你推荐营养均衡、简单易做的早餐方案。

## When to Run

- 用户告知冰箱里有哪些食物（如："冰箱里有鸡蛋、牛奶、面包..."）
- 用户主动询问早餐推荐（如："明天早餐吃什么"、"推荐个早餐"）
- 用户要求设置定时任务（如："每天晚上8点提醒推荐早餐"）
- 用户动态更新食材（如："买了xxx"、"吃完了xxx"、"用掉了xxx"）
- 用户设置饮食偏好（如："我不吃香菜"、"我对花生过敏"）

## Workflow

### 场景1：更新食材清单

当用户告知冰箱里有哪些食物时：

1. 解析用户提供的食材列表
2. 询问是否需要记录保质期（可选）
3. 将食材列表写入 `references/fridge.md`（完整覆盖）
4. 确认更新成功，用表格展示当前食材概览

### 场景2：推荐早餐

当用户询问早餐推荐时：

1. 读取 `references/fridge.md` 获取当前食材
   - 如果文件为空或只有模板内容，提示用户先提供食材清单
2. 读取 `references/preferences.md` 获取用户偏好（如存在）
3. 读取 `references/history.md` 获取近期推荐记录，避免重复
4. 检查是否有即将过期的食材（3天内），优先使用
5. 根据推荐原则生成营养均衡的早餐方案
6. 按照输出格式展示推荐结果
7. 将本次推荐追加到 `references/history.md`（保留最近30条）

**食材不足时的处理**：
- 如果食材种类太少（<3种），告知用户并给出"极简方案"
- 如果缺少某类食材（如没有蛋白质），明确指出并建议补充

### 场景3：设置定时任务

当用户要求定期推荐时：

1. 使用 automation 工具创建定时任务
   - 默认时间：每天 20:00
   - rrule: `FREQ=DAILY;BYHOUR=20;BYMINUTE=0`
2. 任务内容：自动执行早餐推荐流程
3. 推荐完成后记录到历史

### 场景4：动态更新食材

当用户提到以下关键词时触发：

| 关键词 | 操作 |
|--------|------|
| 买了、新增、补货了、囤了 | 添加食材 |
| 吃完了、没有了、用完了、用掉了、消耗了 | 删除食材 |
| 坏了、扔了、过期了 | 删除食材 |
| 还剩xxx | 更新数量 |

执行流程：

1. 读取 `references/fridge.md` 获取当前食材列表
2. 根据用户描述进行增删改操作
3. 更新最后修改时间
4. 写回 `references/fridge.md`
5. 用表格展示变动情况：

```
| 操作 | 食材 | 变动 |
|------|------|------|
| 新增 | 鸡蛋 | +10个 |
| 删除 | 牛奶 | 已用完 |
```

### 场景5：设置饮食偏好

当用户说"我不吃xxx"、"我对xxx过敏"、"我喜欢xxx"时：

1. 读取 `references/preferences.md`（不存在则创建）
2. 更新对应的偏好设置
3. 确认更新成功
4. 后续推荐将自动遵循这些偏好

### 场景6：查看即将过期食材

当用户问"有什么快过期的"或系统检测到过期食材时：

1. 读取 `references/fridge.md`
2. 筛选 3 天内到期的食材
3. 按紧急程度排序展示
4. 给出优先使用建议

## Output Format

推荐早餐时使用以下格式：

```
明天早餐推荐

主食：xxx（做法简述）
蛋白质：xxx（做法简述）
蔬菜/水果：xxx
饮品：xxx

预计准备时间：xx 分钟
小提示：xxx（营养亮点或注意事项）

---
消耗食材：鸡蛋x2、面包x2、牛奶200ml
```

**极简模式**（食材不足时）：

```
极简早餐方案

当前可用食材有限，推荐：xxx

建议补充：蛋白质类（鸡蛋/牛奶）、主食类（面包/燕麦）
```

## Recommendation Principles

遵循以下原则进行推荐：

1. **营养均衡**：确保主食 + 蛋白质 + 蔬菜/水果的组合
2. **简单易做**：优先推荐 15 分钟内可完成的方案
3. **优先消耗**：
   - 即将过期的食材（3天内）
   - 数量较多需要消耗的食材
4. **保持多样性**：避免与近 3 天推荐重复（查看 history.md）
5. **季节适宜**：根据当前月份考虑温热/清爽搭配
   - 冬季（11-2月）：温热为主（热粥、煎蛋、热饮）
   - 夏季（6-8月）：清爽为主（沙拉、冷饮、水果）
   - 春秋：均衡搭配
6. **尊重偏好**：严格遵守用户设置的饮食限制和过敏信息
7. **时间适配**：
   - 工作日：推荐快速方案（<10分钟）
   - 周末：可推荐稍复杂的方案（15-20分钟）

## File Structure

| 文件路径 | 说明 | 自动创建 |
|----------|------|----------|
| `references/fridge.md` | 冰箱食材列表（含保质期） | 是 |
| `references/history.md` | 历史推荐记录，保留最近 30 条 | 是 |
| `references/preferences.md` | 用户饮食偏好设置 | 按需创建 |

### fridge.md 格式

```markdown
# 冰箱食材清单

最后更新：2026-03-13

## 蛋奶类
- 鸡蛋 x 10
- 牛奶 1L（到期：2026-03-15）⚠️
- 酸奶 3盒（到期：2026-03-20）

## 主食类
- 吐司面包 1袋（到期：2026-03-16）
- 燕麦片 500g

## 蔬果类
- 番茄 x 3
- 香蕉 x 4
- 生菜 1颗（到期：2026-03-14）⚠️

## 肉类
- 培根 200g（到期：2026-03-18）

## 其他
- 芝士片 6片
- 花生酱 1罐
```

说明：
- `⚠️` 标记表示 3 天内到期
- 保质期为可选项，用户不提供则不记录

### history.md 格式

```markdown
# 早餐推荐历史

## 2026-03

- 03-13: 番茄鸡蛋三明治 + 牛奶 | 消耗：鸡蛋x1、番茄x1、面包x2、牛奶200ml
- 03-12: 燕麦粥 + 香蕉 + 煎蛋 | 消耗：燕麦50g、香蕉x1、鸡蛋x1
- 03-11: 培根煎蛋 + 吐司 + 牛奶 | 消耗：培根50g、鸡蛋x1、面包x2、牛奶200ml
```

### preferences.md 格式

```markdown
# 饮食偏好设置

## 过敏/禁忌（严格遵守）
- 花生
- 海鲜

## 不喜欢（尽量避免）
- 香菜
- 皮蛋

## 偏好
- 喜欢甜口早餐
- 工作日要快速（<10分钟）
- 喜欢热食

## 特殊需求
- 低糖饮食
```

## Limitations

此 Skill 不适用于：

- 正餐/晚餐推荐（仅限早餐场景）
- 外卖点餐建议
- 餐厅推荐
- 食材购买/采购清单生成
- 复杂烹饪教程（仅提供简单做法说明）
