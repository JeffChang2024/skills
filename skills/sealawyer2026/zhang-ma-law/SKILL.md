# 并购重组专家 🤝

> 并购重组全流程法律服务 - 从尽职调查到交割执行的专业支持

## 🎯 服务领域

### 股权收购
- **资产收购**：资产剥离、债务处理、人员安置
- **股权收购**：股权转让、增资扩股、股权置换
- **合并分立**：吸收合并、新设合并、公司分立
- **借壳上市**：壳资源筛选、重组方案、审批流程

### 尽职调查
- **法律尽调**：主体资格、资产权属、重大合同
- **财务尽调**：财务数据、税务合规、关联交易
- **业务尽调**：商业模式、核心竞争力、市场前景
- **风险识别**：或有负债、诉讼仲裁、行政处罚

### 交易架构
- **支付方式**：现金收购、股权支付、承债收购
- **交易结构**：直接收购、间接收购、VIE架构
- **融资安排**：并购贷款、过桥融资、夹层融资
- **税务筹划**：所得税优化、增值税处理、印花税

### 交割执行
- **交割条件**：先决条件、付款节点、股权过户
- **过渡期管理**：损益归属、重大事项决策
- **对赌安排**：业绩承诺、补偿机制、退出条款
- **投后整合**：治理结构、人员整合、文化融合
