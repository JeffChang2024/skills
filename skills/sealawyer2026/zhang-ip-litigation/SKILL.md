# 知识产权诉讼专家 ⚖️

> 专利/商标/著作权侵权诉讼 - 从侵权分析到赔偿计算的全流程支持

## 🎯 专业领域

- **专利诉讼**：发明专利/实用新型/外观设计侵权判定
- **商标诉讼**：商标侵权/不正当竞争/驰名商标保护
- **著作权诉讼**：作品侵权/盗版维权/网络传播权
- **商业秘密**：技术秘密/经营秘密泄露维权
- **集成电路**：布图设计专有权保护

## 📋 核心服务

### 1.1 侵权分析诊断
```
你是一位资深的知识产权诉讼专家，拥有20年专利、商标、著作权诉讼经验。

请对以下知识产权纠纷进行专业分析：

【权利基础】
- 权利类型：[专利/商标/著作权/商业秘密]
- 权利证书号：[号码]
- 权利有效性：[有效/无效/待定]

【侵权事实】
- 被控侵权行为：[具体描述]
- 侵权产品/服务：[描述]
- 侵权范围：[地域/规模]

请提供：
1. 侵权判定分析（全面覆盖原则/等同原则适用）
2. 赔偿金额估算（法定赔偿/实际损失/侵权获利）
3. 证据保全建议（公证购买/网页存证/现场勘验）
4. 诉讼策略建议（管辖选择/时机把握/被告策略）
5. 风险预警与应对方案
```

### 1.2 诉讼文书起草
```
请根据以下信息起草知识产权诉讼文书：

文书类型：[起诉状/答辩状/上诉状/证据清单/代理词]
案件类型：[专利侵权/商标侵权/著作权侵权/不正当竞争]

【当事人信息】
原告：[名称/地址/法定代表人]
被告：[名称/地址/法定代表人]

【案件要素】
涉案权利：[专利号/商标注册号/作品名称]
侵权事实：[简明描述]
诉讼请求：[具体金额及停止侵权等]

要求：
1. 严格遵循《民事诉讼法》格式要求
2. 侵权比对详细具体
3. 赔偿计算有数据支撑
4. 引用相关司法解释和判例
```

## 📋 核心功能

### 1.1 智能咨询
```
你是一位资深知识产权诉讼专家。请根据以下情况提供专业建议：

案件/项目背景：[描述]
当事人/企业情况：[描述]
核心诉求：[描述]
争议焦点：[如有]

请提供：
1. 法律问题分析
2. 可行方案建议（2-3个）
3. 各方案优劣对比
4. 风险提示
5. 下一步行动建议
```

### 1.2 文书起草
```
请根据以下信息起草法律文书：

文书类型：[类型]
当事人信息：[信息]
案件/项目情况：[描述]
具体诉求：[诉求]

要求：
1. 格式规范
2. 法律依据准确
3. 事实陈述清晰
4. 论证逻辑严密
5. 语言专业得体
```

### 1.3 合同审查
```
请审查以下合同，识别风险点：

[粘贴合同内容]

审查要点：
1. 主体资格审查
2. 标的条款审查
3. 价款/报酬条款
4. 履行条款审查
5. 违约责任审查
6. 争议解决条款
7. 其他风险点

输出格式：
- 🔴 高风险：[说明]
- 🟡 中风险：[说明]
- 🟢 建议优化：[说明]
```

### 1.4 案例分析
```
请分析以下案例/项目：

案情/项目简介：[描述]
争议焦点/关键问题：[描述]
已有证据/材料：[描述]

分析内容：
1. 法律关系梳理
2. 争议焦点分析
3. 有利/不利因素
4. 类似案例参考
5. 策略建议
6. 预期结果评估
```

### 1.5 法规检索
```
请检索与以下问题相关的法律法规：

问题描述：[描述]
适用领域：[领域]
效力层级：[法律/行政法规/部门规章/地方性法规/司法解释]

输出：
1. 核心法律依据（条文引用）
2. 相关司法解释
3. 部门规章/规范性文件
4. 地方性规定（如有）
5. 最新立法动态
```
